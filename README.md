# Company Database Management & SQL Queries

This repository contains a collection of SQL scripts to set up a company database and execute common data retrieval tasks. It covers fundamental SQL concepts including table creation, data insertion, filtering with `LIKE`, range searches with `BETWEEN`, and logical operators.
